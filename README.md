# 🔑 PassManage - By ViratPod

**PassManage** is a modern, lightweight password manager built with **React**, **Vite**, and **TailwindCSS 4.0**. It allows users to securely store and manage passwords locally in the browser, with a sleek, responsive dashboard, real-time notifications, and unique IDs for each entry.  

---

## 🌟 Features

- **Secure Local Storage:** All passwords are saved locally in the browser. No backend required.  
- **Unique IDs:** Each password entry gets a unique ID using **UUID**.  
- **Notifications:** Instant success/error feedback using **react-toastify**.  
- **Responsive UI:** Modern design powered by **TailwindCSS 4.0**.  
- **Easy Management:** Add, edit, delete, and search your passwords easily.  

---

## 🛠️ Technology Stack

- **Frontend:** React + Vite  
- **Styling:** TailwindCSS 4.0  
- **Notifications:** react-toastify  
- **Unique IDs:** UUID  
- **Storage:** localStorage
